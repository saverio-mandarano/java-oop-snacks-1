## Snack 1: Studente

1. Scrivi una classe Student con i seguenti attributi:

- nome
- cognome
- età

2. Implementa un costruttore che accetta tre parametri per inizializzare questi attributi.
3. Aggiungi un metodo che restituisca una stringa concatenata dei valori (es. Anselmo Bitta, 24 anni).

## Snack 2: Gestione di un Conto

1. Crea una classe ContoBancario con attributi per numero di conto e saldo.
2. Implementa un costruttore che accetta il numero di conto e inizializza il saldo a zero.
3. Aggiungi metodi pubblici per depositare denaro sul conto, prelevare denaro dal conto e ottenere il saldo corrente.

## Snack 3: Registro Studenti

1. Crea una classe RegistroStudenti che rappresenti un registro di studenti.
2. Utilizza un array privato per memorizzare oggetti Studente (creati nello Snack 1).
3. Implementa un costruttore senza parametri per inizializzare il registro vuoto e un metodo pubblico per aggiungere studenti al registro.
4. Aggiungi un metodo che stampi la lista degli studenti, senza utilizzare metodi helper.
